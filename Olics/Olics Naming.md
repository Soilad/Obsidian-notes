- Alcohols:
    - **Common:** Alkyl group + "alcohol" (e.g., methyl alcohol).
        
    - **IUPAC:** "Alkanol" (replace 'e' of alkane with 'ol'). Number longest chain from end nearest -OH. For polyhydric, retain 'e' and add di/tri/etc. before 'ol' (e.g., ethane-1,2-diol). Cyclic alcohols use "cyclo-" prefix, -OH at C-1.
        
- Phenols:
    - **Common/IUPAC:** Phenol.
        
    - Substituted: o-, m-, p- (common) or 1,2-, 1,3-, 1,4- (IUPAC).
        
    - Dihydroxy derivatives: Benzene-1,2-diol (catechol), Benzene-1,3-diol (resorcinol), Benzene-1,4-diol (hydroquinone/quinol).
        
- Ethers:
    - **Common:** Alkyl/aryl groups in alphabetical order + "ether" (e.g., ethyl methyl ether). Use "di-" if groups are same.
        
    - **IUPAC:** Hydrocarbon derivative where H is replaced by -OR or -OAr. Larger R group is parent hydrocarbon. Named as "alkoxyalkane" (e.g., methoxybenzene for anisole).
